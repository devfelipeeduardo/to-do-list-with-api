
# ToDoList

Uma lista de tarefas, desenvolvida para estudo.

## Stack utilizada

**Front-end:**

React ✅
Redux ⏳

**Back-end:**

C#/.NET ✅
EntityFramework ✅
JWT ⏳


## Funcionalidades

- Temas dark e light
- Criação de listas de tarefas
- Responsividade Desktop e Mobile


## Instalação

Instale my-project com npm

```bash
  npm install my-project
  cd my-project
```

Run

```bash
npm run dev
```
